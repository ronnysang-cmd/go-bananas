## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Open your browser and go to: `http://localhost:3000`

## Development

For development with auto-restart:
```bash
npm run dev
```

## Usage

1. Click the upload area or drag & drop an image file
2. Click "Upload Photo" to save it
3. View all your uploaded photos in the gallery below

Your photos are stored in the `uploads/` folder.# go-bananas
